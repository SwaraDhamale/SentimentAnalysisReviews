# Sentiment Analysis of Text Reviews

## Project Overview
This project implements sentiment analysis on text reviews using Natural Language Processing (NLP) and Machine Learning techniques. The goal is to classify reviews into Positive, Negative, or Neutral categories.

## Technologies Used
- Python
- Pandas
- Numpy
- NLTK
- Scikit-learn
- Streamlit (optional web app)

## Steps Performed
1. Data collection and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature extraction using TF-IDF
4. Model training (Naive Bayes, SVM, Logistic Regression)
5. Model evaluation using accuracy, precision, recall, F1-score
6. Web application for sentiment prediction (optional)
